# Customer Tracker CRUD App

## Technologies Used
### Server Side
- Node.js
- Express.js
- MySQL (Cloud hosted through JawsDB)
### Client Side
- Handlebars.js
- jQuery
- CSS
### Deployment
- Heroku

---

## Notable Features
- Does not use a Third Party ORM
- Performs all 4 basic CRUD functions
- Follows MVC Structure
